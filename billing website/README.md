# Restaurant Billing Website

A complete restaurant billing system with menu management, shopping cart, billing, and sales reporting.

## Features

- ✅ Menu Display with Images
- ✅ Add to Cart Functionality
- ✅ Menu Management (Add/Edit/Delete)
- ✅ Shopping Cart with Quantity Control
- ✅ QR Code Payment
- ✅ Print Bill
- ✅ Monthly Sales Report
- ✅ Mobile Responsive Design

## Quick Deploy to Netlify

1. Visit: https://app.netlify.com/drop
2. Drag and drop this entire folder
3. Your site will be live in seconds!

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `script.js` - All functionality
- `images/` - Image directory (optional, uses external images by default)

## Usage

1. Open `index.html` in a browser
2. Default menu items are pre-loaded
3. Click items to add to cart
4. Use "Manage Menu" to add/edit items
5. View sales reports in "Sales Report" section

All data is stored in browser localStorage.
