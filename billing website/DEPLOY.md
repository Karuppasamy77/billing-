# Deployment Guide

## Quick Deploy to Netlify (Recommended - 2 minutes)

1. Go to https://app.netlify.com/drop
2. Drag and drop your entire project folder onto the page
3. Wait 10-20 seconds
4. Your site is LIVE! You'll get a URL like: https://random-name-123.netlify.app

## Alternative: GitHub Pages

1. Create account at https://github.com
2. Create new repository
3. Upload files and enable GitHub Pages in settings
